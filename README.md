# 📚 Help Database Project (v1.0)

Projeto de banco de dados desenvolvido para a disciplina de **Projeto Conceitual de Dados**, com o objetivo de modelar um sistema acadêmico para gestão de alunos, professores, cursos e matrículas em instituições de apoio pedagógico.

## 📌 Descrição

O projeto propõe o desenvolvimento de um banco de dados relacional para gerenciamento de processos acadêmicos, incluindo cadastro de alunos, cursos, professores e matrículas.

O sistema foi projetado seguindo as etapas clássicas de modelagem de banco de dados:

- Especificação do **Minimundo**
- **Projeto Conceitual (Diagrama ER)**
- **Projeto Lógico (Modelo Relacional)**
- **Projeto Físico (MySQL)**

## 🗂 Estrutura do Banco

O banco de dados é composto pelas seguintes tabelas principais:

- `Aluno`
- `Area`
- `Curso`
- `Matricula`
- `Modulo`
- `Professor`
- `Topico`

As relações entre tabelas são definidas por **chaves primárias**, **chaves estrangeiras** e **restrições de integridade referencial**.

## ⚙️ Tecnologias Utilizadas

- **MySQL**
- **MySQL Workbench**
- **Modelo Relacional**

## 📊 Funcionalidades do Sistema

O banco permite:

- registrar alunos
- cadastrar cursos e áreas de conhecimento
- organizar módulos e tópicos de cursos
- vincular professores aos cursos
- registrar matrículas de alunos
